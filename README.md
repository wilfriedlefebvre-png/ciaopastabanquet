# Ciao Pasta Banquet Builder

A React application for building banquet proposals with menu selection, guest counts, beverage options, and automatic price calculations.

## Features

- Select from 5 different menu options with varying prices
- Configure adult and kid guest counts
- Choose beverage packages (All beverages, Wine & Beer only, or Non-alcoholic only)
- Optional champagne toast between entr√©e and dessert
- Automatic calculation of subtotal, tax (7.75%), and gratuity (20%)
- Download banquet summary as a text file
- Email proposal directly from the app
- Real-time price summary sidebar

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Create a production build:
```bash
npm run build
```

### Preview Production Build

Preview the production build:
```bash
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
